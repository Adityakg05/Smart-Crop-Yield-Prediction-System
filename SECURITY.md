# Security Policy

## Supported Versions

We only support the latest version of the Smart Crop Yield Prediction System. Please ensure you are running the latest version from the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

We take the security of our project seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, follow these steps:

1. Send an email to the project maintainer at `adityakgupta05@gmail.com`.
2. Include a detailed description of the vulnerability, steps to reproduce it, and any potential impact.
3. Give us reasonable time to investigate and address the issue before making any public disclosure.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

Thank you for helping us keep this project secure!
